# Admin App I18N Glossary

This glossary defines core terms for admin-facing UI text in app.js.

## Core Terms

| Concept | DE (Admin UI) |
|---|---|
| Badge ID | Badge-ID |
| Badge PIN | Badge-PIN |
| Visitor Pass | Besucherkarte |
| Worker Card | Baustellen-Ausweis |
| Day Pass | Baustellen-Tagesausweis |
| Subcompany | Subunternehmen |
| Visitor Company | Besucherfirma |
| Host Contact | Ansprechpartner |
| Visit End | Besuchsende |

## Consistency Rules

- Use Besucherkarte, not Besucherausweis.
- Use Subunternehmen consistently in labels, dialogs, and exports.
- Keep Badge-ID and Badge-PIN unchanged across all screens.
- For day pass visuals, use Baustellen-Tagesausweis.

## Scope

- Applies to admin strings in app.js and related rendered templates.
- Worker app strings are maintained in branding/i18n-glossary-worker.md.

# Worker App I18N Glossary

Use these core terms consistently in worker-facing translations.

## Core Terms

| Concept | DE | EN | TR | AR |
|---|---|---|---|---|
| Badge ID | Badge-ID | Badge ID | Rozet ID | رقم البطاقة |
| Visitor Pass | Besucherkarte | Visitor Pass | Ziyaretci Karti | بطاقة الزائر |
| Worker ID Card | Mitarbeiterausweis | Employee ID | Calisan Kimligi | هوية العامل |
| Turnstile | Drehkreuz | Turnstile | Turnike | البوابة الدوارة |
| PIN | Badge-PIN | Badge PIN | Rozet PIN | رمز PIN للبطاقة |

## Style Rules

- Keep product names untranslated: BauPass, QR, PIN.
- Prefer short, action-oriented button labels.
- Use sentence case for notices and errors.
- Avoid mixing "visitor card" and "visitor ID" in the same locale.

## Notes

- This glossary is for worker app UI texts in worker-app.js and worker.html.
- If a new key is added, pick terms from this table first.
